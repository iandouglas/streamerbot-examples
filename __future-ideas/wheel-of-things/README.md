Inspired by something I saw on PixStitch's stream today, a wheel of choices for things like "lego steps" or "levels of a mario game" etc, where each wheel spin would work toward a number of other goals and onces those goals were met, the item was removed from the wheel.

Could also be used for things like "take a shot of hot sauce" with a limit of 5 shots taken.

This would need some overlay work as well for the animated spinner. Maybe have a choice of what kind of spinner, like a wheel, or a roll.

I'm thinking for Streamer.bot setup I'd like to pull things in from the CSV setup that I've done previously, and then a setting for which kind of display, and make the overlay use a setting from Streamer.bot to display/activate differently via JavaScript/CSS.

The CSV setup will allow a user to set a start/end goal that either counts up or down in the overlay, so  shots of hot sauce would start at 5 and count down to 0, so the overlay would show '3 shots left" and "levels of a game" could start at 0 and go up to, say, 15, and the overlay would show something like "3/15 levels to play" ... the strings might need to be pulled in from the CSV and transmitted to the overlay via streamer.bot

Ideally we'd make a streamdeck button to spin the wheel, or a chat !command to spin the wheel.
